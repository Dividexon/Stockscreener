# Stock Screener - Matrix Style

Ein Echtzeit-Aktien-Screener im Matrix-Stil.

## Features

- **Matrix-Design**: Teal-Text auf schwarzem Hintergrund mit fallenden Zeichen
- **DAX**: Deutsche Blue-Chip-Aktien (SAP, Siemens, Allianz, etc.)
- **Tech-Aktien**: US-Tech-Giganten (Apple, Microsoft, Google, Tesla, etc.)
- **Krypto**: Top-Kryptowährungen (Bitcoin, Ethereum, Solana, etc.)
- **Auto-Refresh**: Daten werden alle 60 Sekunden aktualisiert
- **Responsive Design**: Funktioniert auf Desktop und mobilen Geräten

## Angezeigte Informationen

- **Symbol**: Ticker-Symbol der Aktie
- **Name**: Vollständiger Unternehmensname
- **Preis**: Aktueller Kurs
- **Änderung**: Absolute und prozentuale Kursänderung
- **Status**: Markt offen/geschlossen

## Nutzung

Öffne `index.html` in einem Browser oder besuche die GitHub Pages URL.

## Hinweis

Die Yahoo Finance API hat CORS-Einschränkungen. Bei GitHub Pages wird ein Demo-Modus mit simulierten Daten angezeigt. Für Echtzeit-Daten lokal testen oder einen CORS-Proxy verwenden.

## Technologien

- HTML5 / CSS3 / JavaScript (Vanilla)
- Yahoo Finance API
- GitHub Pages Hosting
